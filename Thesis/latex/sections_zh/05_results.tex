\subsection{域内贯穿距离预测精度与不确定度}
\label{sec:in-domain-accuracy-zh}

表~\ref{tab:model-summary} 总结了三个学习阶段的作用和留出集表现。Stage~1 在每个工况只使用一条代表性曲线进行训练——即直接为每个工况挑选其在 \SI{5}{ms} 处的 q1 外推贯穿距离恰好落在该工况中位数的单条轨迹（对应确定的一个喷孔、确定的一次重复）——充当稳定的均值锚点；其测试 MAE 为 \SI{2.91}{mm}，在对应的代表性测试集上测得。相比之下，Stage~2 解决的是更困难的任务：它在完整过滤归档上同时跨越工况内的 plume-to-plume 差异与 shot-to-shot 差异（同一工况下同一 plume 的各次重复以独立样本的形式进入训练）并学习条件方差；其测试 MAE 为 \SI{5.63}{mm}，是在更大、更异质的测试群体上度量的。两组数字因此并不直接可比：较高的 Stage~2 MAE 反映的是更难的监督目标，而非精度下降，这也使得 Stage~2 成为合适的不确定性感知教师模型。

在保守的未删失 CDF 协议（域内主要逐点指标）下，采用部署蒸馏损失形式（均值 MSE 与对数方差 MSE 的混合，方差权重 \(\lambda_\sigma=5.0\)）在五个种子（\(\{7,17,42,99,2024\}\)）上训练的选定无锚点阶段三 MLP 筛查 surrogate，在每个种子的 542{,}565 个留出时间点上达到整体 RMSE~\(\SI{4.265}{mm}\pm\SI{0.040}{mm}\)、MAE~\(\SI{3.032}{mm}\pm\SI{0.048}{mm}\) 以及微小的正向偏差~\(\SI{+0.562}{mm}\pm\SI{0.119}{mm}\)。作为含删失的补充检查，同一模型在全清洗 CDF 集合（每个种子的 24{,}316 条轨迹、692{,}942 个留出时间点）上得到 RMSE~\(\SI{4.735}{mm}\pm\SI{0.037}{mm}\)；这些清洗后的 CDF 轨迹中有 9,630 条为右删失轨迹，对应 SCR 为 39.6\%，因此把最后一个可见点当作物理终点会引入实质性的协议误差，而不是单纯的记录细节。在主要未删失 CDF 子集上，图~\ref{fig:pred-vs-actual-best} 表明，在整个测量范围内，留出集预测基本贴近恒等线，且随着贯穿距离增大，散点逐步变宽。这种变宽与异方差设计是一致的：模型并没有强行对所有时刻和工况使用同一个固定误差条。

不确定度校准在工程上同样有意义。最终 5 种子平均 1$\sigma$ 与 2$\sigma$ 覆盖率在未删失协议下分别为 88.7\% 和 98.9\%（含删失全集为 86.7\% 和 98.5\%），均明显比名义高斯值 68.3\% 和 95.4\% 更保守。图~\ref{fig:stage3-calibration-coverage} 进一步把这一点展开为按预测标准差分箱的可靠性图和覆盖率曲线，而不只比较平均标准差与平均误差。对于早期设计筛查而言，这种保守偏置有用，但不能被过度解读为经过认证的假阴性安全保证：它使模型不太容易低估晚期时间或弱观测区间下的贯穿距离不确定度，但有害液态撞壁仍需要发动机代表性验证。

\subsection{工况级检查：观测中位数与拟合延续}

在逐点 CDF 指标之外，另外两个协议进一步锐化了 SVGP 与 MLP 的对比。在包含 4{,}213 个点的工况级 P50 集合上，直接拟合 P50 观测的 oracle q1 达到了 \SI{1.010}{mm} 的 RMSE 下限（不可部署）；在学习模型中，SVGP 领先（RMSE \SI{2.649}{mm}），选定 MLP 紧随其后（\(\SI{2.848}{mm}\pm\SI{0.059}{mm}\)），而面向外推正则化的诊断 KD 变体表现最差（\(\SI{3.212}{mm}\pm\SI{0.245}{mm}\)）。然而，在观测到的 P50 窗口之外包含 21{,}527 个点的 q1 外推网格上，排序发生了反转：诊断 KD 变体取得了最低的 RMSE（\(\SI{10.358}{mm}\pm\SI{0.106}{mm}\)）和最小的正向外推偏差（\SI{+6.269}{mm}），领先于 SVGP（\SI{10.591}{mm}，偏差 \SI{+6.410}{mm}）和选定 MLP（\(\SI{10.864}{mm}\pm\SI{0.064}{mm}\)，偏差 \SI{+6.753}{mm}）。这两种 MLP 变体因此探究了一个真正的权衡：选定无锚点模型是一个保守的分布内代理模型，而诊断 KD 变体被保留为一种受外推正则化的版本，在观测窗口外更接近 SVGP 风格的平滑先验。逐种子的详细数值及运行路径见附录~\ref{app:implementation_mapping}。

\subsection{留一喷嘴（LONO）转移性能与分布外失效}
\label{sec:lono-transfer-zh}

\paragraph{方法论说明：双协议评估。}
\label{sec:dual-protocol-zh}
本章使用的两种评估协议——下文报告的 LONO 交叉验证以及后文第~\ref{sec:kd-form-fix-zh} 节报告的五种子生产 bootstrap——是有意区别开的，回答的是不同问题。LONO 协议在训练时留出一个喷嘴，测量模型对训练中从未见过的喷嘴几何的泛化能力，因此报告的是模型的\textbf{域外（OOD）}行为。生产协议在全数据集上使用五个随机种子重新训练同一配置，并在非右删失测试集上评估；它测量的是模型的\textbf{域内拟合质量}及其对随机初始化的敏感性，是核心表中方差报告的合适协议。

这两个协议并列报告而不合并为单一排名，因为各自隔离了独立的论断。一个在 LONO 协议下胜出但在生产 bootstrap 下落败的模型，意味着泛化好但域内拟合差；反之，在生产 bootstrap 胜出但 LONO 落败的模型，意味着域内拟合紧密但对外推至新喷嘴几何的表现差。生产协议支持 MLP 作为测量设计空间内的保守筛查代理；LONO 协议则表明 SVGP 替代主干在留一喷嘴转移的点精度上更强。这些是互补的论断，而不是对某一架构的全面胜利宣告。

留一喷嘴评估用于检查模型是否仅仅在训练期间已见的喷嘴之间进行插值。在锁定阶段二教师为仅 $\mu$-锚点配置并采用刷新后的五折 LONO 协议（留出喷嘴 Nozzle0、Nozzle1、Nozzle2、Nozzle4、Nozzle5）下，LONO 胜出的阶段三 MLP 变体（不确定区间混合原始/KD 监督）的五折平均 RMSE 为 \(\SI{12.55}{mm}\pm\SI{12.46}{mm}\)，MAE 为 \(\SI{9.80}{mm}\pm\SI{10.18}{mm}\)，平均 1$\sigma$/2$\sigma$ 覆盖率为 0.575/0.770。同一未删失 LONO 评估测得阶段三 SVGP 的 RMSE 为 \(\SI{10.16}{mm}\pm\SI{9.57}{mm}\)，MAE 为 \(\SI{7.87}{mm}\pm\SI{7.55}{mm}\)，覆盖率为 0.628/0.821。留一喷嘴评估的结论因此是有条件的：SVGP 是点转移更强的主干，而 MLP 的保留依据是其域内的保守覆盖、喷射起始辅助输出以及消融可负担性（按 §4 开头的术语约定理解）。

极高的标准差主要由 Nozzle0 这一折驱动，它是整个 campaign 中唯一的全新出厂喷嘴。虽然 Nozzle1 的各项几何参数与 Nozzle0 相同，但 Nozzle1--8 均来自不同的喷油器，存在未建模的喷油器间变异；这一结构性差异也解释了为何 Nozzle1--8 的控制背压仅覆盖 4\,bar 层级（而 Nozzle0 为 1\,bar 和 4\,bar），以及为何在不包含 Nozzle0 折时 LONO 协议会崩溃。Nozzle0 因此最好被理解为一个跨设计族失效的外推案例，而非普通的平均性能折：阶段三 MLP 在此折的 RMSE 达到 \SI{34.76}{mm}，SVGP 的 RMSE 达到 \SI{26.99}{mm}，二者均存在系统性低估。如果排除 Nozzle0，MLP 的四折平均 RMSE 降至 \(\SI{7.00}{mm}\pm\SI{1.18}{mm}\)，而 SVGP 达到 \(\SI{5.95}{mm}\pm\SI{1.98}{mm}\)。此时覆盖率的对比交错：1$\sigma$ 覆盖率几乎相同（MLP 0.696 对 SVGP 0.697），而 MLP 的 2$\sigma$ 覆盖率更高（0.923 对 0.890）。因此 LONO 证据反驳了 MLP 全面占优的主张，转而把域内保守部署与跨设计族转移精度分开评估。

\subsubsection{困难工况的视觉证据与失效模式剖析：以 Nozzle 3 为例}

留一喷嘴（LONO）评估显示，代理模型在诸如 Nozzle~3 等特定喷嘴上出现了覆盖率下降与误差放大。为观察这些困难工况在原始数据中的形态，本文将 Phantom 高速 Mie 散射图像与对应的逐喷油束贯穿距离曲线进行了交叉比对（图~\ref{fig:failure-mode-nozzle3}）。

图像序列（$t \approx 0.6 \sim 1.0\,\si{ms}$）显示出明显的孔间非对称性：当左侧喷油束已建立稳定的连续射流时，右下方的三束喷油（约 3 至 5 点钟方向）却显得短小、断续。其中部分喷孔先挤出少量低动量、脱离主体的先导液团，并在环境气阻下迅速减速；这一视觉观察对应于贯穿距离曲线（如 Plume~2 的绿线、Plume~0 的蓝线）在同一时间窗内出现的``平顶''特征。随后（$t > 1.0\,\si{ms}$），在同一方位出现了主射流并迅速反超先导液团，使曲线呈现阶跃式陡升。

这种``先导液团 + 反超主射流''的轨迹形态，与分段或多段喷射文献中报道的尾迹驱动追赶/反超机制一致：先到达的液团可通过尾迹效应和轴向气动量预处理改变其后液团的贯穿距离。因为本实验既无同步气速测量，也无内部流动可视化或液团身份追踪，该案例只能被解释为与此类机制``相一致的''图像证据；针阀偏摆或瞬态喷孔节流等机械原因同样是现有数据所不能排除的候选解释。

数据所能直接支持的，是建模层面的推论。降阶拟合目标（$k_{1/4}t^{1/4}$）按构造单调且呈凹形，无法包络这类二阶导数发生突变的阶跃轨迹。将平滑单调先验强加于这类轨迹会造成目标失真，这是导致 Stage~2、Stage~3 网络在受影响喷嘴上覆盖率下降的一个可识别原因。要对此类极端瞬态喷雾进行建模，因此需要采用更柔性的形状先验，或者在能够分辨真实机制的诊断手段就位后，对该机制进行显式的条件化。

\subsection{神经代理模型消融研究}
\label{sec:ablation-study-zh}

前文章节确立了跨架构结果：SVGP 目前在点精度上是赢家（尤其在 LONO 转移协议下），而选定 MLP 则是域内更为保守的筛查代理模型。接下来的证据解释了为何 MLP 技术栈仍被保留作为运行工作流：它的训练课程可以被廉价地消融，其喷射起始辅助输出支持 GUI 原型，且它的区制感知蒸馏使得保守筛查行为具有可追溯性而非偶然。

\subsubsection{Stage-2 anchor 调度消融实验}
\label{sec:stage2-anchor-ablation-zh}

阶段二在异方差 NLL 之上引入了两个可选的早期 anchor 惩罚项：把 \(\mu(t{\to}0)\to 0\) 的二次惩罚（权重 \(\lambda_\mu\)）以及把超出固定下限 \(\sigma_{0}\) 的过多标准差 \(\max(0,\,\sigma(t{\to}0)-\sigma_{0})\) 拉小的二次惩罚（权重 \(\lambda_\sigma\)）。两者均与三角核 \(w(t)=\max(0,\,1-t/\tau)\) 加权积分，作用在 \(\tau=\SI{0.15}{ms}\) 的早期时间窗口。为检查这些 anchor 究竟是帮助还是损害跨喷嘴泛化，本文在与阶段一相同的 LONO 协议下对比了三个阶段二变体：(i)~无 anchor（\(\lambda_\mu=\lambda_\sigma=0\)）；(ii)~仅 $\mu$-anchor（\(\lambda_\mu=10^{-2}\)，\(\lambda_\sigma=0\)）；(iii)~$\mu$+$\sigma$ 双 anchor（\(\lambda_\mu=10^{-2}\)，\(\lambda_\sigma=10^{-3}\)）。三个变体共用阶段一胜出的生产七特征配置（$\alpha=0.5$ 带压力残差）作为热启动，共用相同的五折 LONO 划分、相同的阶段三无锚点蒸馏链、相同的网络架构与优化器、以及单一固定种子（42）；唯一变化的轴是阶段二的 anchor 配置。本节中的留出集指标是在完整的阶段一$\to$二$\to$三流水线之后，基于留出喷嘴的未截断 CDF~P50 点进行评估的（各折评估点数依次为：Nozzle0 311{,}149、Nozzle2 60{,}369、Nozzle5 33{,}017、Nozzle4 28{,}139、Nozzle1 25{,}705）。

该消融实验检验两个耦合的假设。首先，零时刻的均值 anchor 是否能在不造成其他位置条件均值偏差的代价下注入有用的物理先验；无 anchor 与仅 $\mu$-anchor 的对比孤立了这一权衡。其次，把早期 \(\sigma\) 强行压向其下限是否会损害对留出喷嘴的不确定度标定；仅 $\mu$-anchor 与 $\mu$+$\sigma$ 双 anchor 的对比则把这一轴单独剥离开来，不受均值 anchor 的混杂影响。

两个假设的验证都指向支持仅 \(\mu\)-anchor 的结论。激活 \(\mu\)-anchor 会使平均 RMSE 降低 \SI{1.81}{mm}（从 14.54 降至 \SI{12.73}{mm}），MAE 降低 \SI{0.94}{mm}，2\(\sigma\) 覆盖率从 0.763 提升至 0.775，并将系统性低估偏差从 \SI{-8.50}{mm} 收窄至 \SI{-7.11}{mm}。改进主要集中在 Nozzle2 折：在没有 anchor 时，NLL 可以随意将数据稀少且靠近 \(t{=}0\) 的窗口用作松弛空间，导致无 anchor 在此处崩溃至 \SI{19.18}{mm} 的 RMSE；anchor 移除了这一自由度。这三个来自同一测试系列的喷嘴（Nozzle1/4/5，每折约 25k--33k 评估点）在三个变体下的结果彼此相差都在 \SI{1}{mm} 以内，可见 anchor 的价值只有在 LONO 折真正构成 OOD 挑战时才会显现。

在顶部叠加 \(\sigma\)-anchor 会把平均预测标准差从 \SI{5.68}{mm} 进一步收紧到 \SI{5.49}{mm}，但并未产生补偿性增益：RMSE 反而上升到 \SI{13.05}{mm} 且 2\(\sigma\) 覆盖率降至 0.764。这一现象可被解释为：将 \(\sigma\) 在 \(t{=}0\) 附近推向下限，会经由阶段三蒸馏传播为一种更自信但略带偏差的 \(\mu\) 预测，从而同时损害精度和标定。因此，下一小节报告的阶段三消融实验将阶段二教师固定在仅 $\mu$-anchor 配置下（\(\lambda_\mu=10^{-2}\)，\(\lambda_\sigma=0\)）；任何关于阶段三区制加权或 KD 形式的结论，都以该 anchor 选择为前提。完整的逐折明细与原始聚合文件归档路径见附录~\ref{app:implementation_mapping}。

\subsubsection{测试阶段3中原始数据与教师模型的不同混合比例}

在阶段二 anchor 消融之后，本文将阶段二教师固定为仅 $\mu$-anchor 配置，并在同一五折 LONO 协议下重新运行阶段三的消融。该消融对比了四种阶段三配置：
(i)~\textbf{基准配置}，在可靠时间箱中设定 $w_{\mathrm{kd}}=0.25$ 且 $\lambda_{\mathrm{anchor}}=0.01$；
(ii)~\textbf{仅可靠区间纯原始变体}，移除可靠区间中的 KD 混合；
(iii)~\textbf{不确定区间混合变体}，在不确定区间中重新引入部分原始监督；以及
(iv)~\textbf{无锚点变体}，在保留 raw/KD 区制设计不变的同时设定 $\lambda_{\mathrm{anchor}}=0$——由于阶段二教师已经固定为仅 $\mu$-anchor 配置，且 $\lambda_{\mathrm{anchor}}=0$ 本就折算进了基准 raw/KD 区制划分之中，该变体在数值上与表~\ref{tab:stage2-anchor-ablation} 中的 \texttt{mu\_anchor} 行完全相同，此处列出仅为保持四种阶段三区制对比的完整自洽。
表~\ref{tab:stage3-ablation-ranking} 提供了 LONO 排序。不确定区间混合变体取得了最低的平均 RMSE 和 MAE，而无锚点变体取得了最高的 2\(\sigma\) 覆盖率。各变体之间的差距较小：最佳和最差的平均 RMSE 仅相差 \SI{0.21}{mm}，远低于大约 \SI{12.5}{mm} 的跨折标准差。因此，这应当被解读为在阶段三调参上的微弱偏好，而不是决定性的架构级影响。

各折的模式解释了这个较小的总体差距。Nozzle0 对每个阶段三变体来说仍然是一个跨设计族的外推失败，RMSE 在 \SIrange{34.76}{35.31}{mm} 的范围内，并存在约 \(\approx\SI{-27}{mm}\) 的系统性低估偏差。不确定区间混合变体在 Nozzle2 和 Nozzle1 折上获胜，而基准配置在 Nozzle5 和 Nozzle4 折上获胜。因此，表~\ref{tab:stage3-ablation-ranking} 支持在固定阶段二仅 $\mu$-anchor 教师下将不确定区间混合变体作为 LONO 的 RMSE/MAE 胜出配置，但这并不构成对 70\%/20\% 覆盖率边界、\(\sigma_{\mathrm{ref}}=\SI{10}{mm}\) 或所有可能权重组合的全面超参数搜索。完整的运行工件归档路径见附录~\ref{app:implementation_mapping}。

\subsubsection{修复视野边界附近的系统性低估问题}
\label{sec:kd-form-fix-zh}

切换到将对数压力残差直接作为输入的"幅度缩放加压力残差"特征变体后，根据近视场指标 \(\mathrm{pen}\ge 0.9\,\mathrm{cap}_\mathrm{nozzle}\) 对阶段三的残差切片，暴露出高贯穿距离区间中系统的性能下降：尽管阶段二教师在接近 FOV 顶部（cap）时几乎无偏（bias \SI{-0.59}{mm}，RMSE \SI{4.82}{mm}），但蒸馏出的无锚点学生在同一切片上的表现恶化为 bias \SI{-2.70}{mm} 且 RMSE 为 \SI{6.11}{mm}。这种漂移无法归因于阶段一/二的拟合——所采用的稳健清洗流程（Hampel 滤波与单调差分下界检测，详见 §3）在 cap 附近并未导致教师向下漂移——因此只可能由阶段三蒸馏步骤内部产生。

为定位机制，本文在保持上述教师不变的基础上，执行了七组单变量消融：关闭原始 NLL、关闭一阶/二阶形状惩罚、将 forward KL 中的置信度权重 \(\sigma_{\mathrm{ref}}/\sigma_{\mathrm{teacher}}\) 强制为 1、将 forward KL 替换为 reverse KL、使用仅 \(\mu\) 的 MSE 替换完整 KL、将训练缩短到 10 轮、以及将不确定与仅教师两个区制中的 KD 权重翻倍。其中两个变体较为突出：关闭原始 NLL 的诊断变体几乎精确重现了教师在近 cap 处的行为（bias \SI{-0.53}{mm}）；将完整 KL 替换为仅均值 MSE 的诊断变体把近 cap 偏差拉回至 \SI{-0.91}{mm}，并在所有变体中取得了全域最低 RMSE。二者指向两个共同原因：(i)~原始 NLL 把学生 \(\mu\) 拉向位于 FOV 顶部截断下一帧的下界观测值；(ii)~forward KL 项 \((\mu_t-\mu_s)^{2}/\sigma_s^{2}\) 允许学生仅靠膨胀自身的 \(\sigma\) 就可以逃避匹配教师 \(\mu\)。其余五个变体均把近 cap 偏差留在 \([-2.7,-2.0]\) mm 范围内，从而被排除为主要机制。

仅均值 MSE 的诊断变体切断了这一均值失配下的方差膨胀路径，但同时丢失了 KL 对学生 \(\sigma\) 的隐式监督，使得 KD-only 区间（不确定区间与仅教师区间）内的 \(\sigma\) 标定漂移。因此本文提出了均值 MSE 与对数方差 MSE 的混合蒸馏损失
\begin{equation}
\mathcal{L}_{\mathrm{KD}} \;=\; (\mu_s-\mu_t)^{2} \;+\; \lambda_\sigma\,\bigl(\log\sigma_s^{2} - \log\sigma_t^{2}\bigr)^{2},
\label{eq:kd-mse-mu-plus-sigma}
\end{equation}
即在均值 MSE 之上增加一个显式的对数方差 MSE，使 \(\sigma\) 重新受到监督，但同时避免重新引入 KL 形式所允许的均值失配下方差膨胀。通过对 \(\lambda_\sigma\in\{0.5,1,2,5\}\) 进行扫描，选定 \(\lambda_\sigma=5.0\) 作为阶段三新默认值：在此设置下，近 cap 处的 RMSE 和 bias 达到联合最佳（\SI{4.89}{mm} 与 \SI{-0.66}{mm}），统计上与教师的 \(\SI{-0.59}{mm}\) 无法区分；同时在仅教师区间切片上逐点 \(|\sigma_s-\sigma_t|\) 中位值降至 \SI{0.27}{mm}。图~\ref{fig:kd-overlay} 叠加了教师、基准 forward-KL 学生与新混合蒸馏学生的偏差--真值曲线；改善主要集中在贯穿距离的较高区间，下半段保持未受干扰。

\subsubsection{方法论补充：消融可负担性作为紧凑神经网络架构的特有优势}
\label{sec:mlp-ablation-affordability-zh}

选择 MLP 作为贯穿距离代理模型架构的一个直接结果，是本文所报告的系统化消融在普通硬件上计算可负担。本文中独立的消融矩阵共有三组：阶段一在 9 个工程化特征变体上的 5 折 LONO（第~\ref{sec:stage1-lono-ablation} 小节）、阶段二在 3 种 anchor schedule 上的同协议 LONO 对比、以及阶段三在 4 种区制加权变体与 7 组单变量 KD 形式消融上的组合搜索。支持头表所需的完整 LONO~$\times$~消融~$\times$~种子网格，在单张消费级加速器上大约消耗数十 GPU 小时。

更核心的论证不在单次训练耗时，而在任务结构与架构归纳偏置的匹配关系。本文代理模型的目标是把 $(\mu,\sigma)$ 作为低维工况描述子（时间、缩放后的幅值、对数压力残差、喷嘴独热指示）的逐点函数进行回归，输入既无规则空间网格、也无长程序列依赖。激发 CNN 设计的局部平移不变性偏置和驱动 Transformer 的长程注意力偏置都与此种逐点低维回归任务结构不匹配；两者会引入任务并不需要的表达能力与参数量。在此基础上，本文设计归因所要求的特征工程、anchor schedule、区制权重、KD 损失形式的组合扫描——共数十种配置~$\times$~五折 LONO~$\times$~多个种子——需要在每个变体上独立完整训练；引入更复杂的时空主干会同时放大单次成本与可比配置数量，结果是硕士级别算力预算下只能报告单一配置而无法分离各设计选择的边际贡献。因此，架构-任务匹配与消融可负担性共同使 MLP 成为有原则性的选择，而不是仅仅为省事而做出的权宜之计；本文第~4 章与第~5 章呈现的系统化消融证据，若没有这种匹配，在硕士级别算力预算下是无法触及的。这一定性结论不依赖对 CNN/Transformer 训练成本的具体数值断言，本文也未对此进行直接基准测试。

\subsection{可选后续：残差喷嘴头自适应}
\label{sec:residual-family-head-zh}

\emph{本节是一个可选的后训练后续分析；它探索可扩展性与喷嘴自适应，但不改变前文的部署模型结论——MLP/SVGP 权衡与部署选择保持不变。其最强结果仅为域内结果，未经 LONO 检验。}上文确立的筛查代理模型通过输入特征中的喷嘴 one-hot 指示符对喷嘴进行条件化。前述各节遗留了两个问题，促成这次可选分析。其一，\textbf{部署与可扩展性}：one-hot 喷嘴输入对一个全新喷嘴没有干净的结构化答案，只能依赖硬编码的 fallback 身份或重新训练一个头。其二，\textbf{族内自适应}：能否在不放弃已部署主干的前提下，使同一套 censoring-aware 协议在支持的喷嘴设计族内进一步适配？本节报告一个自洽的研究性后续，在完全相同的保守 uncensored CDF 点表（\num{542565} 点；协议见第~\ref{sec:evaluation-protocol-en} 小节）上评估，\emph{并不}修改上文已确立的部署模型结论。

所有变体共有的构造是\textbf{在冻结、热启动的骨干之上叠加逐喷嘴加性残差}：
\begin{equation}
\mu(x,f) = \underbrace{\mu_{\mathrm{shared}}(h(x))}_{\text{frozen, hot-started}} + \underbrace{\Delta_f(h(x))}_{\text{trained, zero-initialised}}, \qquad \Delta_f \equiv 0 \ \text{at init.}
\label{eq:residual-family-head-zh}
\end{equation}
共享骨干从已部署的 Stage~3 检查点复制并冻结；仅训练小的逐喷嘴残差头与对数方差头（可训练参数 \num{49923}，冻结骨干参数 \num{351361}）。由于 $\Delta_f$ 在初始化时为零，未见过的喷嘴会回退到共享模型——一个无需硬编码身份的结构性 fallback；收缩先验（对 $\Delta_f$ 施加 $L_2$ 惩罚，权重 $\lambda_\Delta$）在逐喷嘴拟合与靠近共享映射之间做权衡。

\subsubsection{通过更丰富的喷嘴适配器实现精度逐步提升}
\label{sec:residual-climb-zh}

从残差头出发，在相同协议下评估了同一“冻结骨干 + 零初始化残差”原则的两个更强版本；表~\ref{tab:residual-climb} 汇总了这一爬升。\emph{（Step~1）}残差头把可扩展的逐喷嘴头部署从 \SI{5.010}{mm} 改进到 \SI{4.631}{mm} CDF RMSE，校准同步改善（ECE $0.072\to0.060$，CRPS $2.72\to2.38$），工况级 P50 从 \SI{3.93}{mm} 降到 \SI{3.40}{mm}；收缩 sweep $\lambda_\Delta\in\{0,10^{-4},10^{-3},10^{-2}\}$（图~\ref{fig:residual-delta-sweep}）显示每个设置都超过部署头，其中 $10^{-4}$ 在主 RMSE 上最优。\emph{（Step~2）}逐喷嘴调制最后一层主干表示的特征逐元素线性调制（FiLM，Feature-wise Linear Modulation）适配器（$h' = \gamma_f \odot h + \beta_f$，恒等初始化），而非只对输出做偏移，把 RMSE 进一步降到 \SI{4.510}{mm}、ECE 降到 $0.040$——表明喷嘴信息部分是表示性的，而不仅是输出偏移，此时已低于部署的 one-hot 模型。\emph{（Step~3）}把 MLP 主干换成加性残差 multi-task SVGP——共享稀疏 GP 从已部署检查点热启动，逐喷嘴残差过程带零残差 fallback，喷嘴身份保持在核特征向量之外——达到 \SI{3.987}{mm} CDF RMSE，把校准误差减半至 ECE $0.026$，并把 P50 降到 \SI{2.08}{mm}。在这一可选分析内部，这是主表上最强的域内后续结果：比论文最强主线预测器（SVGP，\SI{4.193}{mm}）低 \SI{0.206}{mm}（$\approx$5\%），P50 低 \SI{0.57}{mm}。这 \SI{3.987}{mm} 仅为域内结果：residual multi-task SVGP 未经 LONO 协议检验（表~\ref{tab:residual-lono} 仅覆盖 MLP 残差臂），其跨喷嘴泛化能力尚未验证。图~\ref{fig:residual-svgp-climb} 展示了 residual-SVGP 相对此前最优的增益。

\subsubsection{鲁棒性：LONO 转移与数据敏感性}
\label{sec:residual-robustness-zh}

\subsubsection{残差适配器的适用边界：Nozzle6 与 Nozzle0}
\label{sec:residual-boundary-zh}

残差的胜势受喷嘴设计族的限制，两个案例标出了这条边界。\textbf{Nozzle6} 是一个数据稀疏（\num{20758} 点）的 fold，所有 MLP 臂都坍缩到 \SIrange{12}{14}{mm}；残差臂在此比纯喷嘴头还差 $\approx\SI{2.3}{mm}$，并系统性低估（偏差 $\approx-\SI{10.5}{mm}$）——属过度专用化而非转移。\textbf{Nozzle0} 是唯一的第一代喷油器（BC2022）；即便做了动量归一化，其 early-time 贯穿响应仍约为改型喷嘴的 2.5 倍且始终不低于后者（图~\ref{fig:injector-comparison}），因此占据了冻结主干从未表示过的 early-time 包络（图~\ref{fig:injector-scatter} 将该分支分离）。完全留出时，N0 是真正的 zero-shot 情形：残差代理得到 \SI{33.16}{mm} RMSE，与第~\ref{sec:svgp-lono-comparison-zh} 小节报告的部署版 MLP 在 Nozzle0 上 \SI{34.76}{mm} 的 LONO 失效一致。

一项 few-shot 研究量化了廉价逐族修正能把 N0 恢复到何种程度（图~\ref{fig:n0-fewshot}）。冻结整个 N0-留出模型、仅微调 N0 残差头于 $k$ 个 N0 工况上，RMSE 从 \SI{33.16}{mm}（$k=0$）下降，但即便用上全部 203 个工况也停在 $\approx\SI{15.7}{mm}$——远高于同一架构在族内 LONO fold 上达到的 $\approx\SI{5}{mm}$。两个效应限制了它。其一，冻结的对数方差头在超出设计族的 N0 特征上严重失配，扼制了 NLL 微调，使其停在比绕开它的 MSE 拟合高 $\approx\SI{4}{mm}$ 处；因此可部署的 adapter 必须同时适配方差头。其二，即便均值拟合不被扼制，delta 头能移除全局偏差（$k=2$ 时 $-27\to\approx0$ mm），却无法收紧残差离散——逐族输出偏移无法合成主干从未学到的表示。因此一个超出设计族的喷油器需要主干级容量（解冻、跨所有主干块的 FiLM，或重训），而不只是一个新的输出头。

总之，这一可选后续表明，冻结骨干上的加性残差可以改善喷嘴设计族内的自适应。Residual multi-task SVGP 在主 uncensored CDF 表上给出了最强域内后续结果（\SI{3.987}{mm} RMSE，\SI{2.08}{mm} P50，ECE $0.026$），在点误差与校准两方面都优于论文最强主线预测器；部署与可扩展性的缺口被零初始化残差及其自动的共享模型 fallback 在\emph{喷嘴设计族内}结构性地闭合；MLP 的 LONO 弱点也被部分修复（族内约 31\%）。Nozzle6 与 Nozzle0 两个案例诚实地划出边界：廉价 adapter 在设计族内有效，并会在一个真正全新的喷嘴族需要主干级容量时发出信号，而非将其掩盖。本扩展是一次后训练研究性后续——LONO 转移只在 MLP 残差臂上测量，因此 residual-SVGP 的跨喷嘴转移行为尚未确立——它并不改变前述各节的部署模型结论；MLP/SVGP 主结果仍以前文报告为准。

\subsection{原型筛查输出与证据边界}
\label{sec:prototype-screening-zh}

最终 surrogate 可以与一个简化的壁面接近筛查几何耦合。当前的 GUI 原型使用 MLP，因为它已经暴露出时间筛查显示所需的喷射起始（onset）辅助输出；SVGP 可以替代贯穿距离的 $\mu,\sigma$ 通道，但它本身不能直接替代该起始预测头。在默认的玩具输入下：
\[
\left\{
\begin{aligned}
d_n &= \SI{0.34}{mm},\\
p_{\mathrm{inj}} &= \SI{2000}{bar},\\
p_{\mathrm{amb}} &= \SI{15}{bar},\\
\theta_{\mathrm{tilt}} &= 20^{\circ},\\
N_{\mathrm{plume}} &= 10,\\
t_{\mathrm{inj}} &= \SI{800}{\micro s},\\
\mathrm{RPM} &= 250,
\end{aligned}
\right.
\]
原型在 0--\SI{5}{ms} 窗口内生成了用于绘图的非负均值 $\mu_{+}(t)\in[\SI{1.70}{mm},\,\SI{112.95}{mm}]$ 和 $\sigma_{\parallel}(t)\in[\SI{4.39}{mm},\,\SI{17.66}{mm}]$。面向排序的瞬时近壁截获分数在 $t=\SI{4.237}{ms}$ 时达到峰值 $P_{\mathrm{hit}}=\SI{77.66}{\percent}$，梯形积分得到的累积几何暴露量为 $E=\SI{2.475}{ms}$。这些数值说明，端到端链路已经能够将贯穿距离分布转化为时间分辨的筛查分数；它们并不代表有害液态壁面润湿的实测概率。图~\ref{fig:impingement-preview-zh} 给出峰值时刻的空间快照，图~\ref{fig:impingement-summary-zh} 总结了 \SI{5}{ms} 窗口内的概率几何时间历程。

证据边界同样重要。上述截获分数只应被解读为管道的演示和面向排序的几何 surrogate 指标，不能直接等同于真实发动机中发生有害液态撞壁的实际概率。本文验证过的贡献是其测量范围内的贯穿距离 surrogate 以及它与概率几何筛查接口的耦合，而不是对有害液态撞壁进行相态分辨的预测器。本文报告的数值声明均可追溯至清洗后的轨迹表、保存的检查点，以及项目归档中所包含的导出评估图。

\subsection{本章小结：回到评估地图}
\label{sec:results-chapter-summary-zh}

本章开篇即把每一个不同的评估论断映射到表~\ref{tab:results-claims-map-zh} 上，以避免把 uncensored、含删失、工况级与跨喷嘴转移等协议混为一张单一排行榜。在依次梳理完每个协议之后，该表所承诺的每一项论断如今都已各自配上其样本总体、种子数与主要数字得到确立；下面的清单按行回到该地图，并指出每个数字的推导位置。

\begin{itemize}
\item \textbf{域内点精度}：选定 MLP RMSE \SI{4.265}{mm}（覆盖率 0.887/0.989）；SVGP RMSE \SI{4.193}{mm}（第~\ref{sec:in-domain-accuracy-zh} 节）。
\item \textbf{含删失（补充）}：MLP RMSE \SI{4.735}{mm}，P95 \SI{9.167}{mm}（第~\ref{sec:in-domain-accuracy-zh} 节）。
\item \textbf{观测中位数检查}：SVGP RMSE \SI{2.649}{mm} 领先；MLP \SI{2.848}{mm}（第~\ref{sec:condition-level-checks-zh} 节）。
\item \textbf{拟合延续检查}：排序反转，诊断 KD 变体以 \SI{10.358}{mm} 最优（第~\ref{sec:condition-level-checks-zh} 节）。
\item \textbf{留一喷嘴转移（完整 LONO）}：SVGP \SI{10.16}{mm} $<$ MLP \SI{12.55}{mm}；Nozzle0 折对两种架构均崩塌（第~\ref{sec:lono-transfer-zh} 节）。
\item \textbf{族内转移（剔除 Nozzle0）}：MLP \SI{7.00}{mm}，SVGP \SI{5.95}{mm}；覆盖率交织（第~\ref{sec:lono-transfer-zh} 节）。
\item \textbf{课程消融}：$\mu$-anchor 教师加不确定区间混合变体胜出，差距远低于跨折标准差（第~\ref{sec:ablation-study-zh} 节）。
\item \textbf{KD 形式修复}：近 cap 偏差从 \SI{-2.70}{mm} 改善到 \SI{-0.57}{mm}；主要 RMSE \SI{4.265}{mm} 在五个种子间保持稳定（第~\ref{sec:kd-form-fix-zh} 节）。
\item \textbf{残差后续（可选）}：域内 RMSE \SI{3.987}{mm}（未经 LONO 检验）；族内 LONO 转移改进约 31\%（\SIrange{7.44}{5.16}{mm}）（第~\ref{sec:residual-family-head-zh} 节）。
\item \textbf{原型筛查演示}：面向排序的几何截获分数（例如峰值 $P_{\mathrm{hit}}=\SI{77.66}{\percent}$）；并非经过验证的撞壁概率（第~\ref{sec:prototype-screening-zh} 节）。
\end{itemize}

这些结果为接下来两章设定了论述基础。第~6 章从讨论的角度重新审视分阶段 surrogate 设计——将阶段三定位为筛查模型而非第一性原理模型，陈述已验证的发现及其适用边界，并反思随时间变化的标度指数以及概率性撞壁筛查原型的未来验证路径。第~7 章随后回到第~1 章所构建的六阶段流水线，总结每一阶段已验证的内容，并收束全文，附上一张把 MLP/SVGP 权衡浓缩为单一表格的成绩单。
