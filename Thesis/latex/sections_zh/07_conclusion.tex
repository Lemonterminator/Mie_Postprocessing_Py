\section{结论}

本文证明，多孔柴油喷射的大规模 Mie 散射归档视频可以转化为一套经过标定、具备不确定度感知的穿深 surrogate，从而在无需 CFD 的前提下支持早期几何筛查。其架构整合了方法章节中建立的有限视场处理、plume 级异方差性、物理幅值 $A = (\Delta P/\rho_a)^{1/4}d_n^{1/2}$ 缩放，以及原始 CDF/教师蒸馏机制。最终模型在统一评估协议上超越三条基线，保持保守的经验覆盖率，并可泛化至未见喷嘴族，其概率输出可直接投影为逐帧几何筛查分数。

\subsection{已验证结果}

\begin{itemize}
  \item 在 795{,}561 个预测点和 33{,}845 条轨迹的外部清洗评估协议上，三阶段过渡区缩放 MLP（anchor-off）四次重复训练平均达到 RMSE~\SI{6.094}{mm}、MAE~\SI{4.465}{mm}、P95~\SI{12.888}{mm}，并给出 0.783/0.982 的保守 1$\sigma$/2$\sigma$ 覆盖率。
  \item 在较早的原始归档留出切分上，同一 anchor-off Stage~3 surrogate 在 21{,}470 条轨迹上达到 RMSE~\SI{6.68}{mm}、MAE~\SI{4.83}{mm}、bias~\SI{-0.02}{mm}，并给出 72.24\% / 97.03\% 的保守 1$\sigma$/2$\sigma$ 覆盖率。
  \item 相对校准后的 Hiroyasu--Arai、Naber--Siebers 和稀疏异方差 Gaussian process，三阶段模型分别降低 RMSE 38.9\%、31.1\% 和 11.7\%。
  \item 留一喷嘴族评估显示该方法在未见喷嘴族上仍优于物理基线，但困难喷嘴族的低覆盖说明外推风险仍需显式报告。
\end{itemize}

\subsection{当前原型边界}

\begin{itemize}
  \item 概率性 wall-interaction 筛查说明，穿深 surrogate 已经可以被投影为时间分辨的几何风险分数，但当前旋转 Gaussian 几何与玩具活塞案例只是流水线演示，仍未与匹配实验或 CFD 做验证。
  \item 该模型应被理解为非反应、室温、Mie 定义工况域内的早期筛查 surrogate，而不是对真实发动机液态撞壁的相态分辨预测器。
\end{itemize}

\subsection{下一步开发重点}

\begin{enumerate}
  \item 用匹配的光学实验或 CFD 碰撞数据验证 $P_{\mathrm{hit}}(t)$ 与累积暴露量 $E$。
  \item 用可学习的二维包络模型替代当前解析的常锥角包络。
  \item 量化多次训练重复中的 seed-to-seed 可复现性和标定稳定性。
  \item 引入喷嘴族条件化的瞬态先验，使 HZ\_Nozzle3 和 Nozzle0 这类困难案例得到更明确的建模。
\end{enumerate}
