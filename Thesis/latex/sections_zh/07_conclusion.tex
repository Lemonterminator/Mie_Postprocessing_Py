\section{结论}

第 1 章将本文设定为一条端到端承载的六阶段流水线——数据提取、删失感知整理、具有物理依据的目标构造、分阶段不确定度感知训练、多协议评估，以及筛查部署；本章回到这一序列，说明每一阶段已验证的内容及原型边界所在。

本文证明，多孔柴油喷射的大规模 Mie 散射归档视频可以转化为一套经过标定、具备不确定度感知的贯穿距离 surrogate，从而在无需 CFD 的前提下支持早期几何筛查。本文的贡献是右删失感知的学习协议——有限视场处理、plume 级异方差性、具有物理依据的幅值 $A$ 缩放，以及原始 CDF/教师蒸馏——以及概率性撞壁接口，而不是承载它的具体神经网络主干。最终的分阶段 MLP 在统一评估协议下以大幅度超越两个物理基线，并在分布内不确定度覆盖率上明显比 SVGP 基线更保守，但 SVGP 在分布内和留一喷嘴评估的逐点精度上都是更强的回归器。因此 MLP 主干被保留下来，是因为它具有 onset-CDF 输出头、regime 感知的蒸馏链，以及第四章、第五章设计归因证据所依赖的消融可负担性；而分阶段 surrogate 应被理解为右删失感知学习协议的一种 regressor-agnostic 实例化：同一协议同样可以承载核均值函数；上文的 LONO 结果（平均 RMSE \(\SI{10.16}{mm}\) 对 \(\SI{12.55}{mm}\)）表明，在同一协议下核均值函数有望提升分布外精度。

\begin{itemize}
  \item 在保守的无删失 CDF 协议（域内主要逐点指标）下，生产三阶段 MLP（anchor-off，\texttt{kd\_mse\_mu\_plus\_sigma} 损失，\(\lambda_\sigma=5.0\)，五种子 7/17/42/99/2024）相对在同一协议下重新标定的 Hiroyasu--Arai（\(\SI{10.192}{mm}\)）与延迟 Naber--Siebers（\(\SI{9.212}{mm}\)）基线（第五章）分别降低 RMSE 59\% 和 55\%（表~\ref{tab:conclusions-scorecard-zh}）。在全清洗集合（额外包含右删失点）上的含删失补充评估见第五章。
  \item 分布内 SVGP 是更强的逐点回归器，但 MLP 在不确定度覆盖率上明显更保守（1$\sigma$/2$\sigma$ \(0.888/0.990\) 对 \(0.717/0.946\)）；MLP 相对 SVGP 的结构性价值因此在于保守覆盖率而非点精度。
  \item 在留一喷嘴转移下，SVGP 在五折中四折赢得点 RMSE；剔除两架构都会崩塌的脱离设计族 Nozzle0 折（MLP RMSE \(\sim\SI{35}{mm}\)，SVGP \(\sim\SI{27}{mm}\)）后，1$\sigma$ 覆盖率差距闭合（\(0.696\) 对 \(0.697\)）。相对遗留 LONO 参考物理基线（Hiroyasu--Arai \(\SI{10.713}{mm}\pm\SI{1.542}{mm}\)，Naber--Siebers \(\SI{9.784}{mm}\pm\SI{2.102}{mm}\)；第五章），情况是分裂的：在完整五折均值上 Nozzle0 折的崩塌占主导，因此 MLP（\(\SI{12.55}{mm}\)）高于两个基线，而 SVGP（\(\SI{10.16}{mm}\)）低于 Hiroyasu--Arai 但高于 Naber--Siebers；只有在剔除脱离设计族的 Nozzle0 折之后，两架构才双双明显低于两个基线（MLP \(\SI{7.00}{mm}\)，SVGP \(\SI{5.95}{mm}\)）。这正是第~\ref{sec:kernel-vs-nn-ood-zh} 节所讨论的归纳偏置特征；困难喷嘴的低覆盖说明对真正脱离设计族几何的外推风险仍需显式报告。
\end{itemize}

\subsection{当前原型边界}

\begin{itemize}
  \item 概率性 wall-interaction 筛查说明，贯穿距离 surrogate 已经可以被投影为时间分辨的几何风险分数，但当前旋转 Gaussian 几何与玩具活塞案例只是流水线演示，仍未与匹配实验或 CFD 做验证。
  \item 该模型应被理解为非反应、室温、Mie 定义工况域内的早期筛查 surrogate，而不是对真实发动机液态撞壁的相态分辨预测器。
\end{itemize}

\subsection{下一步开发重点}

\begin{enumerate}
  \item 用匹配的光学实验或 CFD 碰撞数据验证 $P_{\mathrm{hit}}(t)$ 与累积暴露量 $E$。
  \item 用可学习的二维包络模型替代当前解析的常锥角包络。
  \item 按 recording、工况和喷嘴聚类报告工况级置信区间，并给出 bootstrap 置信带，因为 540{,}367 个逐点样本在同一条轨迹内部存在时间相关性，不能视为相互独立。
  \item 引入喷嘴条件化的瞬态先验，使 Nozzle~3（尾迹驱动的 slug/阶梯式贯穿距离）等同族内困难案例，以及脱离设计族的 Nozzle0，都能得到更明确的建模。
\end{enumerate}
