\section{结论}

本文证明，多孔柴油喷射的大规模 Mie 散射归档视频可以转化为一套经过标定、具备不确定度感知的穿深 surrogate，从而在无需 CFD 的前提下支持早期几何筛查。本文的贡献是右删失感知的学习协议——有限视场处理、plume 级异方差性、具有物理依据的幅值 $A$ 缩放，以及原始 CDF/教师蒸馏——以及概率性撞壁接口，而不是承载它的具体神经网络主干。最终的分阶段 MLP 在统一评估协议下以大幅度超越两个物理基线，并在分布内不确定度覆盖率上明显比稀疏异方差 Gaussian process 基线更保守，但 SVGP 在分布内和留一喷嘴族评估的逐点精度上都是更强的回归器。因此 MLP 主干被保留下来，是因为它具有 onset-CDF 输出头、regime 感知的蒸馏链，以及第四章、第五章设计归因证据所依赖的消融可负担性；而分阶段 surrogate 应被理解为右删失感知学习协议的一种 regressor-agnostic 实例化：同一协议同样可以承载核均值函数，并很可能在分布外的精度上从中受益。

\subsection{已验证结果}

\begin{itemize}
  \item 在刷新后的全清洗 CDF 评估协议上（692{,}942 个预测点、24{,}316 条轨迹，五次重复训练，seeds 7/17/42/99/2024），生产三阶段 MLP（anchor-off，\texttt{kd\_mse\_mu\_plus\_sigma} 损失，\(\lambda_\sigma=5.0\)）达到均值 RMSE \(\SI{4.735}{mm}\pm\SI{0.037}{mm}\)、MAE \(\SI{3.293}{mm}\pm\SI{0.039}{mm}\)、bias \(\SI{+0.626}{mm}\pm\SI{0.112}{mm}\)、P95 \SI{9.167}{mm}，以及 0.867/0.985 的保守 1$\sigma$/2$\sigma$ 覆盖率。
  \item 将同一五次重复评估限制在保守无删失 CDF 子集（密度骤降和 FOV 饱和截断后剩余 542{,}565 个点）上，得到 RMSE \(\SI{4.265}{mm}\pm\SI{0.040}{mm}\)、MAE \(\SI{3.032}{mm}\pm\SI{0.048}{mm}\)、1$\sigma$/2$\sigma$ 覆盖率 0.887/0.989，证明主要结论不依赖右删失点的存在。
  \item 相对同协议下校准后的 Hiroyasu--Arai（RMSE \SI{11.268}{mm}）和延迟 Naber--Siebers（RMSE \SI{10.545}{mm}），三阶段 MLP 分别降低 RMSE 58.0\% 和 55.1\%。稀疏异方差 Gaussian process 是更强的逐点回归器（RMSE \SI{4.628}{mm}），但 1$\sigma$/2$\sigma$ 覆盖率仅 \(0.703/0.939\)，对照 MLP 的 \(0.867/0.985\)；在无删失 CDF 子集上 MLP 的覆盖优势进一步扩大到 \(0.887/0.989\) 对 \(0.716/0.945\)。MLP 在分布内相对 SVGP 的结构性价值因此是保守覆盖率而非点精度。
  \item 留一喷嘴族评估采用与 MLP LONO 消融相同的无删失 CDF 点集；MLP 五折均值 RMSE \SI{12.55}{mm}，SVGP 为 \SI{10.16}{mm}。排除两架构都会崩塌的脱离设计族 Nozzle0 折（MLP RMSE \(\sim\SI{35}{mm}\)，SVGP \(\sim\SI{27}{mm}\)）后，MLP 四折均值收窄至 \SIrange{7.0}{7.2}{mm}，SVGP 为 \SI{5.95}{mm}。两架构均明显低于物理基线 LONO 参考值（Hiroyasu--Arai \(\SI{10.713}{mm}\pm\SI{1.542}{mm}\)，Naber--Siebers \(\SI{9.784}{mm}\pm\SI{2.102}{mm}\)），但 SVGP 在五折中四折赢得点 RMSE，并且四折 1$\sigma$ 覆盖率差距已闭合（\(0.697\) 对 \(0.696\)）。这正是第~\ref{sec:kernel-vs-nn-ood-zh} 节所讨论的归纳偏置特征；困难喷嘴族的低覆盖说明对真正脱离设计族几何的外推风险仍需显式报告。
\end{itemize}

\subsection{当前原型边界}

\begin{itemize}
  \item 概率性 wall-interaction 筛查说明，穿深 surrogate 已经可以被投影为时间分辨的几何风险分数，但当前旋转 Gaussian 几何与玩具活塞案例只是流水线演示，仍未与匹配实验或 CFD 做验证。
  \item 该模型应被理解为非反应、室温、Mie 定义工况域内的早期筛查 surrogate，而不是对真实发动机液态撞壁的相态分辨预测器。
\end{itemize}

\subsection{下一步开发重点}

\begin{enumerate}
  \item 用匹配的光学实验或 CFD 碰撞数据验证 $P_{\mathrm{hit}}(t)$ 与累积暴露量 $E$。
  \item 用可学习的二维包络模型替代当前解析的常锥角包络。
  \item 量化多次训练重复中的 seed-to-seed 可复现性和标定稳定性。
  \item 引入喷嘴族条件化的瞬态先验，使 HZ\_Nozzle3 和 Nozzle0 这类困难案例得到更明确的建模。
\end{enumerate}
