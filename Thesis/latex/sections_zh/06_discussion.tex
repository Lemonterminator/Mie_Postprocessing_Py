\section{讨论}

\subsection{为什么 Stage-3 surrogate 是最可信的筛查模型}

Stage~1 是必要的，因为它在引入完整数据集之前提供了稳定的代表性均值轨迹，并验证了 $A$ 缩放假设。但单独使用 Stage~1 会忽略大部分工况内变异，因此不能作为最终筛查模型。

Stage~2 是第一个在完整过滤数据集上学习条件分布的模型。它的作用是把条件均值和条件离散度分开，并为晚期稀疏观测提供教师分布。这正是它的 MAE 略高于 Stage~1 的原因：它面对的是一个更困难、也更贴近真实筛查任务的监督问题。

Stage~3 之所以是最可信的筛查模型，是因为它结合了前两个阶段的优势。在原始观测较密集的地方，它直接从测得的 CDF 数据学习；在观测变得稀疏的地方，它按照第~2 章建立的右删失依据退回到 Stage~2 教师。这种混合监督解释了它在早期时间上的更好保真度，以及在最终结果中表现出的更保守的晚期行为。

\subsection{已验证发现与适用边界}

在本文声明的范围内，有四项结论得到了结果支持。第一，这套光学工作流能够在归档尺度上从多孔 Mie 视频中提取可重复的、面向穿深的目标。第二，降阶拟合阶段同时提供平滑和面向删失观测的实际目标构造。第三，分阶段 surrogate 从代表性均值预测发展到具有不确定度感知的完整数据集预测，并最终达到基于原始 CDF 精化的筛查性能。第四，最终的不确定度区间足够保守，可以支持谨慎的早期工程排序，而不仅仅是点预测。

本文对结论适用范围作出明确限定。
所提取的喷雾边界被解释为特定光学协议下由 Mie 成像定义的液相包络观测量，
而非完整的缸内液相真值 \cite{ecn_liquid_penetration,ecn_liquid_penetration_accessory}。
模型设计主要面向训练工况域内的插值预测，而非无约束外推。
该模型并不旨在替代 CFD、发动机试验或经验证的喷雾--壁面撞击物理模型；
其定位更为具体，即作为一层可审计的早期工程筛查工具，在启动更高保真验证之前收窄设计空间。

另一个限制来自目标定义本身。降阶拟合会收敛到以四分之一次方分支为主导的延拓，而神经 surrogate 又通过正则化被约束为单调递增。这些都是有意施加的形状先验：它们强制轨迹增长符合直觉，并帮助修补观测窗口尾端的短时外推。但它们并不模拟 Mie 散射的成像机理。当光学信号在 plume 到达图像边界之前就减弱时，拟合目标可能通过继续延长穿深轨迹来补偿信号损失。因此，得到的穿深应被解释为 Mie 定义的几何包络延续，而不是强液相喷雾尖端的直接证据。它并不能直接量化局部液体质量、液滴尺寸、相态，或壁膜形成潜力 \cite{ecn_liquid_penetration,ecn_liquid_penetration_accessory}。

这一外推假设目前仍缺少直接对照验证。原计划的验证路径，是使用工业合作方提供的同类型喷嘴和相近工况下的宽视场单喷油束实验视频；若视场约为当前多孔归档的两倍，则可完整观测穿深轨迹直至平台区。随后可人为截断图像视场，仅用截断前时间序列拟合 q1 模型，并将外推曲线与截断后仍可见的完整测量轨迹逐时刻比较。该实验能直接检验本文 KD 链条的核心假设：参数形式 $\sigma\!\left((t-t_0)/s\right)\cdot k_{1/4}\,t^{1/4}$ 在视场边界之外仍是可信延拓，而不是主要由形状先验支配的偏差。由于合作方未能提供此类单喷油束数据，该验证未能在本文范围内完成；因此 q1 外推的可靠时间范围仍是一个开放经验问题。

这一点对下游壁面筛查很重要。喷雾与壁面的相互作用并不总是有害：如果大部分液相在到达壁面之前已经蒸发，剩余的壁面相互作用可能危害更小，且在某些工况下甚至有助于卷吸和混合。真正需要优先关注的是直接的液滴撞壁和持续壁膜形成 \cite{MorieiraMotaPanao2010,liu2024marine_impingement,peraza2022swi,ahmad2025fueldilution}。因此，本文模型更应被理解为一种对 Mie 定义喷雾包络可能接近壁面的保守筛查 surrogate，而不是对有害液态撞壁的相态分辨预测器。

另一个限制来自下游发动机几何耦合。在当前原型中，活塞运动只通过滑块--曲柄运动学平移壁面 mask；它不会通过随时间变化的腔压、压力波动力学或大尺度气流运动反向影响穿深 surrogate。这一简化继承自 OSCC 测量本身，因为每次喷射事件都发生在固定背压的刚性腔体中。因此，该 surrogate 学到的是准静态舱内条件下的穿深，而不是完全耦合的缸内瞬态。由于柴油喷雾穿深对环境气体密度敏感，这种解耦应被理解为一种务实的建模选择，而不是对压力和气流效应可忽略的证明 \cite{NaberSiebers1996}。

但在预定用途内，这种近似仍可被视为一阶工程筛查。本文的目标场景是靠近上止点、时间尺度为毫秒级的短预喷窗口；在当前玩具活塞设置下，报告的 \SI{5}{ms} 窗口内冠面位移小于 \SI{1}{mm}，而当前需求是几何排序，而不是曲轴转角分辨的缸内真值预测。对于低速或中等转速的大缸径双燃料发动机，这种短时间窗内的解耦可以作为保守筛查接受。随着转速上升、喷射时刻远离上止点，或缸内瞬态压力与气流速度显著改变喷雾所见环境状态时，这一近似会变弱。在这些条件下，当前 wall-interaction 分数只能被读作解耦的筛查 proxy，而不是耦合的穿深预测器。

另一个相关限制是喷雾锥角的处理方式。虽然图像处理阶段确实提取了 cone-angle proxy，但本文并未在不同 campaign 之间对其做系统分析，也没有把它提升为最终 surrogate 的学习目标。因此，在下游 wall-interaction 模型中，半锥角 $\alpha$ 是作为固定输入参数出现的，而不是一个经验证的时变观测量。这是一个不可忽略的简化：既有喷雾研究表明，扩张角会随瞬态发展和工况变化而改变，环境气体密度、喷射条件和喷嘴几何都会共同影响其行为 \cite{NaberSiebers1996,JungManinSkeenPickett2015}。因此，当前几何筛查只学习了轴向穿深方向上的不确定度；由 $\alpha$ 决定的横向包络仍然是被假定的，而不是被预测的。

同样需要清楚说明的限制还包括：
\begin{itemize}
  \item 数据是在室温、非反应条件下获得，并依赖于特定光学定义下的液相包络；它们并不验证蒸发、点火或真实壁膜物理。
	  \item surrogate 仅在一个被支持的工况域内训练，不应在真正分布外的外推条件下被赋予同等可信度。
	  \item 下游几何模型使用旋转 Gaussian 包络、简化的活塞运动学和固定半锥角，并把活塞驱动的压力与气流反馈从穿深预测中解耦；它是近壁筛查 proxy，而不是生产级撞壁求解器。
	  \item campaign 级别的差异以及困难的 HZ\_Nozzle3 案例表明，喷嘴族依赖的早期瞬态动力学仍然只被部分捕捉。
	  \item 留一喷嘴族评估表明，分阶段 MLP 在未见喷嘴族上仍低于两个物理基线的 RMSE，但最困难留出族出现低覆盖。因此，跨喷嘴族迁移已经被部分压力测试，但尚不能被视为完全可靠的分布外预测器。
	  \item 外部清洗评估报告了四次 anchor-off 重复训练，seed-to-seed RMSE 标准差为 \SI{0.120}{mm}。该均值比单次点估计更可靠，但仍不等同于完整统计认证，因为尚未报告 bootstrap 置信带。
	  \item 分阶段 surrogate 已在同一外部清洗协议下与校准 Hiroyasu--Arai、延迟 Naber--Siebers 和稀疏异方差 Gaussian-process 基线比较，也与自身 Stage~3 消融比较。但 random forest 或 gradient boosting 等轻量确定性回归器仍未纳入，因此神经网络结构相对这些替代模型的边际价值尚未量化。
	  \item 原始 held-out RMSE 和 MAE 基于 21{,}470 条轨迹中的 519{,}570 个逐点样本计算，但同一轨迹内的时间点误差彼此相关。因此，独立样本数显著小于逐点样本数，任何基于逐点数量的置信表述都应结合这种相关性理解。
	\end{itemize}

\subsection{概率性碰壁筛查原型的未来验证路径}

第~4.7 节所描述的碰壁筛查原型实现了文献综述中所主张的概率性 proxy，但逐帧碰撞概率评分 $P_{\mathrm{hit}}(t)$ 尚未与实物喷雾--壁面观测进行比对。这一空白是有明确定义的后续工作，而非开放性限制，因为所需的参考数据已存在于工业合作方的实验数据集中。

最直接的验证目标是代表性活塞几何下的高速光学实验，即在发动机代表性边界条件下对喷射过程进行逐帧成像。这类实验能直接观测到喷雾在活塞冠附近的扩展情况，从而与 $P_{\mathrm{hit}}(t)$ 进行对比：观测到喷雾接触的首帧给出了碰撞起始的经验边界，而喷射窗口内的累积喷雾暴露量则可与积分评分 $E = \int P_{\mathrm{hit}}\,\mathrm{d}t$ 比较。这一比较将在训练数据集（刚性静止腔体，无活塞运动）无法提供的动态几何条件下，对轴向穿深 surrogate 进行直接检验。

另一条互补验证路径是 CFD 数值模拟。高保真喷雾计算能够分辨液相分布、壁面液膜质量和局部液滴浓度——而 Mie 成像定义的 surrogate 用旋转 Gaussian 包络来近似这些物理量。该 surrogate 的定位并非取代 CFD，而是作为低成本的上游筛查层：碰撞概率超过阈值的工况进入完整计算，安全低于阈值的工况则被过滤掉。验证这一分层工作流——筛查召回率、假阴性率与计算量节省——需要一组 CFD 标注参考工况，而当前数据集尚不具备这些。

两条验证路径都将解决当前原型中一个结构性开放问题：固定半锥角 $\alpha$ 是否是可接受的简化，还是说用第~3 章提取的锥角 proxy 驱动时变横向展宽模型会对碰撞概率结果产生实质影响。只有在获得逐帧匹配的参考数据之后，对 $\alpha$ 的灵敏度才能被可靠评估。
