\section{Methodology Primer for Non-Machine-Learning Readers}
\label{app:ml-primer-en}

This appendix is intended for readers whose primary background is in fluid mechanics or mechanical design. It briefly introduces four categories of techniques involved in this thesis: computer vision, robust optimization, GPU parallel computing, and neural network training. The main text only invokes these concepts when necessary to explain methodological choices; more fundamental terminological explanations are consolidated here to prevent the literature review from assuming a tutorial function.

\paragraph{Computer Vision and Signal Processing.}
Computers store images as numerical matrices, where each pixel corresponds to an intensity value in the matrix. Image processing is essentially performing mathematical operations on these matrices. \emph{Convolution} is one of the most common operations: a small template window slides across the image pixel by pixel, performing a weighted sum at each step. Different convolution kernels can be used for smoothing, edge extraction, or texture enhancement. The Sobel operator is a class of convolution kernels used to estimate local brightness gradients. It responds strongly to abrupt intensity changes such as liquid-phase boundaries, making it suitable for spray boundary localization. The \emph{polar coordinate transformation} converts images from pixel row-column coordinates to a coordinate system with angle and radial distance as axes; by applying this transformation around the nozzle center, the directional structure of a multi-hole spray can be collapsed into a one-dimensional angular signal. The \emph{Fast Fourier Transform} (FFT) decomposes signals into frequency components, making periodic structures appear as frequency-domain peaks; this study utilizes it to identify the rotational periodicity of the multi-hole spray and estimate the nozzle hole angle offset.

\paragraph{Robust Optimization and Curve Fitting.}
Least-squares fitting seeks a curve that minimizes the sum of squared residuals between the observation points and the curve. When a small number of frames are affected by optical noise or quantization errors, the squared penalty amplifies the influence of these points. The Huber loss retains the squared penalty when the residual is small, and transitions to an approximate absolute value penalty once the residual exceeds a threshold, thereby reducing the leverage effect of outliers \cite{Huber1964,NocedalWright2006}. Gradient descent in neural network training calculates the partial derivatives of the loss function at the current parameters and iteratively updates the parameters in the direction that reduces the loss.

\paragraph{GPU Parallel Computing.}
CPUs typically have fewer but more complex cores, suitable for serial logic; GPUs possess a large number of simpler cores, ideal for performing the same operation simultaneously on many data elements. Convolution, thresholding, angular binning, and frame-by-frame aggregation in image processing perfectly fit this paradigm. CUDA is NVIDIA's parallel computing platform; CuPy provides a Python interface similar to NumPy, allowing the processing pipeline in this study to offload the primary array operations to the GPU. Its engineering significance lies in transforming the processing of terabyte-scale video archives from a lengthy task, which is difficult to iterate repeatedly, into a workflow that can be re-run as calibrations, thresholds, and objective definitions are adjusted \cite{NickollsBuckGarlandSkadron2008}.

\paragraph{Neural Networks and Deep Learning.}
The Multilayer Perceptron (MLP) used in this study consists of several linear layers and non-linear activation functions, taking operating condition features and time as inputs to output penetration predictions. Dropout randomly masks a portion of neurons during training, reducing the risk of overfitting under sparse operating condition support \cite{Srivastava2014Dropout}. AdamW decouples weight decay from the adaptive gradient update, helping to improve generalization \cite{LoshchilovHutter2019AdamW}. Smooth activation functions such as SiLU/Swish make derivative transmission more continuous, facilitating the use of physical derivative penalties \cite{Ramachandran2017Swish}. Mean Squared Error (MSE) corresponds to conditional mean regression; heteroscedastic Gaussian Negative Log-Likelihood (NLL) allows the network to output both the conditional mean and conditional variance simultaneously, providing error bands of varying widths for different operating conditions \cite{KendallGal2017}. Knowledge distillation first trains a teacher model that performs stably on clean data, and then uses its soft outputs to guide the student model to learn in regions where labels are sparse or unreliable \cite{HintonVinyalsDean2015}.
